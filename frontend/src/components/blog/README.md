# Blog Components

This folder contains all blog-related components for the salon website.

## Components

- **BlogPage.jsx** - Main blog listing page with search, filtering, and pagination
- **BlogPostDetail.jsx** - Individual blog post detail page with related posts

## Features

- Blog post listing with categories
- Search functionality
- Featured and trending posts
- Related posts suggestions
- Social media integration
- Newsletter subscription
- Responsive design

## Usage

```jsx
import { BlogPage, BlogPostDetail } from './components/blog';
```

## API Integration

These components integrate with the blog API to fetch:
- Blog posts
- Categories
- Author information
- Related posts
- Newsletter subscriptions
